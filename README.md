# Room Booking Web App

Простое веб-приложение для бронирования комнат, созданное на Flask.  
Поддерживает регистрацию пользователей, авторизацию, роли (администратор и пользователь),  
а также полный CRUD-функционал для управления комнатами и бронированиями.

---

## Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/yourusername/room-booking.git
cd room-booking
2. Создайте виртуальное окружение и активируйте его
bash

python -m venv venv
venv\Scripts\activate        # для Windows
# или
source venv/bin/activate     # для Linux/Mac
3. Установите зависимости
bash

pip install Flask werkzeug
4. Инициализируйте базу данных
bash

python database.py
5. Запустите приложение
bash

flask run
После запуска откройте http://127.0.0.1:5000 в браузере.

Данные по умолчанию
Роль	Логин	Пароль
Администратор	admin	admin

Основные возможности
Регистрация и вход пользователей

Роли и права доступа (пользователь / администратор)

CRUD-операции для управления комнатами (добавление, редактирование, удаление)

Бронирование комнат пользователями

Просмотр и отмена броней

Защита от XSS и SQL-инъекций

Инструкция по тестированию
1. Регистрация и вход
Перейдите на /register, создайте нового пользователя.

Попробуйте зарегистрироваться повторно с тем же именем — должно появиться сообщение об ошибке.

Войдите как созданный пользователь (testuser).

Выйдите и войдите как администратор (admin / admin).

2. Проверка ролей
Попробуйте открыть /admin как обычный пользователь — доступ должен быть запрещен.

Войдите под admin — страница должна открыться успешно.

3. CRUD для администратора
Добавьте комнату на панели администратора.

Измените данные комнаты.

Удалите комнату.

Проверьте, что все изменения видны на главной странице.

4. Бронирование
Войдите как обычный пользователь.

Нажмите "Забронировать" у любой комнаты.

Укажите дату и время — появится уведомление об успешной броне.

Перейдите в "Мои бронирования" — бронь должна отображаться.

Отмените бронь — она исчезнет из списка.

Войдите как admin, отмените бронь пользователя через /admin.

5. Безопасность
XSS: Введите <script>alert('XSS')</script> в названии комнаты — текст должен отобразиться как строка, без выполнения скрипта.

SQL Injection: Код использует параметризированные запросы (?), что защищает от SQL-инъекций.
